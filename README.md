# 301pass
此应用程序是一个真正的 301 重定向应用程序，因为它在响应标头中响应 301 状态代码。

# 运行方式

http抢答301,系统ubutu22.04。

测试秘钥：G3HD-MFYQ-8H7J-450Q

使用方法： 下载运行软件：

````
wget https://github.com/xcazt/301pass/raw/refs/heads/main/z301 && chmod 777 /root/z301 && ./z301
````
